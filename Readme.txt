How to run the Online BookStore Shopping Portal using PHP and MySQL

1. Extract the file and copy book folder

2.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name onlineshopping

6. Import onlineshopping SQL database file into your database (given under SQL Database folder)

7.Run the script http://localhost/book 

(frontend)

8.Create your  user account on the sign up page and start to buy or sell books


Admin(Bookstore Manager)

9. Link for admin Panel : http://localhost/book/dashboard.php
